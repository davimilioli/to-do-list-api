# API - Lista de Tarefas (To-do List)

## Sobre a API

Esta é a API backend simples para o projeto To-do List, desenvolvida em Node.js + Express.

- Criar tarefa
- Listar tarefas
- Atualizar tarefa
- Excluir tarefa

## Frontend

Este backend é consumido pelo projeto To-do List (Angular).
- Repositório: https://github.com/davimilioli/to-do-list

## Tecnologias Utilizadas

- Node.js
- Express
- TypeScript
- SQLite

## Funcionalidades

- `GET /todo` → Listar todas as tarefas
- `POST /todo` → Criar nova tarefa
- `PUT /todo/:id` → Atualizar tarefa
- `DELETE /todo/:id` → Excluir tarefa

## Pré-requisitos

- Node.js

## Rodar o projeto

### 1. Instalar dependências
```bash
    npm install
```

### 2. Iniciar ambiente de desenvolvimento

```bash
    npm run start-dev
```
